# Python Mini Projects Collection

⭐ If this repository helps you learn Python, consider starring the project and subscribing to the YouTube channel for more free programming tutorials.

A collection of **25 beginner-friendly Python projects** designed to help new programmers learn Python through hands-on practice.

Each project focuses on a specific programming concept and is accompanied by a complete video tutorial on YouTube.

## How to Run

Clone the repository:

```bash
git clone https://github.com/badrnezhad/Python-Knight-Beginner.git
```

Navigate to the project directory:

```bash
cd Python-Knight-Beginner
```

Run any project:

```bash
python Project-01.py
```

## Recommended Learning Path

Start from Project 01 and continue sequentially. The projects are intentionally organized from simpler concepts to more advanced topics.
